
# Бот для подбора вузов с ИИ

Поступление — всегда стресс. Этот Telegram-бот должен помочь справится с поиском подходящих для вас вузов, исходя из личных требований и возможностей. Он работает прямо в приложении: собирает карту абитуриента (какие предметы сдавал, какие баллы получил, куда хочет поступать и какой бюджет), а затем выдает топ-5 лучших вузов, подходящих по требованиям. 

Результаты открываются в отдельном окне внутри самого Telegram, где можно посмотреть шансы на поступление на каждую специальность, ссылки на ВУЗы и почитать выжимку из реальных отзывов студентов.

## Как это работает

- **Пошаговый опрос:** Сначала пользователь вводит сдаваемые предметы и полученные баллы ЕГЭ, потом выбирает сферы интересов или конкретные специальности, затем финансовые предпочтения и потребность в общежитии.
- **Проверка ИИ:** Перед поиском нейросеть проверяет реальные шансы на поступление на выбранную специальность с теми предметами, что сдал абитуриент.
- **Подбор вузов:** ИИ (Llama 3.3 70b через Groq) подбирает 5 лучших университетов. Для каждого сразу считается вероятность поступления на бюджет и платную основу.
- **Mini App интерфейс:** Результаты выводятся не просто текстом в чат, а через Telegram Web App. Внутри есть удобная навигация по вузам и программам.
- **Суммаризация отзывов:** Не нужно читать сотни комментариев вручную. ИИ парсит отзывы и выдает короткую сводку: главные плюсы и минусы конкретного вуза.

## Технологии

Проект разбит на две части: сам бот и FastAPI сервер, который обслуживает Mini App и делает запросы к LLM.

- **Язык:** Python 3.10+
- **Бот:** Aiogram 3.x
- **Бэкенд:** FastAPI, Uvicorn
- **ИИ:** Groq API (модель `llama-3.3-70b-versatile`)
- **Запросы:** aiohttp
- **Фронтенд Mini App:** HTML, CSS, Vanilla JS
