

🎓 Умный подбор университетов с ИИ

Telegram-бот с интегрированным Mini App, который помогает абитуриентам найти идеальный университет. Бот собирает данные пользователя (баллы ЕГЭ, желаемые специальности, финансовые предпочтения), а нейросеть анализирует информацию и подбирает топ-5 вузов с подходящими образовательными программами. Результаты визуализируются в удобном Telegram Mini App.

✨ Основные возможности

   Пошаговый сбор данных: интуитивный интерфейс бота для ввода баллов ЕГЭ, выбора предметов и предпочтений (город, бюджет/платное, общежитие).
   Двухэтапный выбор профессии: удобный выбор через inline-клавиатуры (сначала общая сфера, затем конкретная специальность).
   ИИ-анализ соответствия: нейросеть проверяет, соответствуют ли сданные предметы требования выбранной специальности.
   Подбор вузов: генерация топ-5 университетов с детальной разбивкой шансов поступления на бюджет и платное отделение.
   Telegram Mini App (SPA): современный интерфейс внутри Telegram для просмотра результатов без необходимости переходить во внешний браузер.
   ИИ-суммаризатор отзывов: анализ реальных отзывов студентов с помощью LLM для формирования краткой выжимки плюсов и минусов вуза.

 🛠 Технологический стек

   Backend & Bot: Python 3.10+, [Aiogram 3.x](https://aiogram.dev/) (асинхронное взаимодействие с Telegram API), [FastAPI](https://fastapi.tiangolo.com/) + Uvicorn (API сервер).
   AI / LLM: [Groq API](https://groq.com/) (модель `llama-3.3-70b-versatile`).
   Frontend: HTML, CSS, Vanilla JS (Telegram Web App SDK).
   Network: `aiohttp` (асинхронные HTTP-запросы к API и парсинг).

 🚀 Установка и запуск

 1. Подготовка репозитория
Склонируйте проект к себе на машину:
```bash
git clone https://github.com/твой_ник/uni-mini-app-bot.git
cd uni-mini-app-bot
```

 2. Настройка окружения
Создайте и активируйте виртуальное окружение:
```bash
python -m venv venv
 Для macOS/Linux:
source venv/bin/activate
 Для Windows:
venv\Scripts\activate
```

Установите необходимые зависимости:
```bash
pip install -r requirements.txt
```

 3. Конфигурация
Создайте файл `.env` на основе примера и заполните его своими ключами:
```bash
cp .env.example .env
```
Вам потребуется указать токен Telegram-бота (получить у @BotFather) и API-ключ Groq.

 4. Запуск проекта
Для работы приложения требуется запустить два процесса (сервер и бота) в разных терминалах.

Терминал 1 — Запуск backend-сервера:
```bash
cd server
uvicorn server:app --reload
```

Терминал 2 — Запуск Telegram-бота:
```bash
cd bot
python bot.py
```

После запуска откройте вашего бота в Telegram и отправьте команду `/start`, чтобы начать подбор университета.
